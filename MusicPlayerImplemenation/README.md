This project is basically me taking what we learned about linked lists and turning it into something that actually feels real, like a music player instead of just storing random numbers.
I built a system where each node holds a Song object, and everything is connected using a doubly circular linked list. That means every song points to the next and the previous one, and the last song loops back to the first so you can keep moving through the playlist forever. There’s no “end” which is kind of the whole point of a playlist.
I started with the Song class, which just stores the title and the singer. Then I made the DNode class, which holds a Song and has pointers to the next and previous nodes. I also used a static variable to keep track of how many nodes exist, which helped me understand what was happening in memory.
After that I built the CircleDLinkedList, which is basically the playlist itself. It uses a cursor to keep track of the current song. From there I implemented functions to add a song, remove a song, move forward, move backward, and print the list. The hardest part here was making sure that every time I added or removed something, both pointers were updated correctly so the structure didn’t break.
Then I created the MusicPlayer class, which is what actually controls everything. This is where I added functions like play, next, previous, addSong, and removeSong. I also implemented a recursive print function to display the playlist forward or backward, which took a little extra thinking to make sure it stopped at the right time.
One thing I had to be really careful with was keeping the list circular. If one pointer is wrong, the whole thing breaks or you end up in an infinite loop. Another important part was handling edge cases, like when there’s only one song in the playlist.
To run the program, I compiled all the files together using g++ and then executed the output file. Once it runs, it shows the playlist behavior by playing songs, moving forward and backward, removing songs, and printing the list in both directions.
Overall, this project helped me actually understand how linked lists work beyond just theory. Seeing how everything connects and how one small mistake can mess up the whole structure made it a lot more clear.
