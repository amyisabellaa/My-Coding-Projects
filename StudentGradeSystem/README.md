~Student Grade System

This project is a student grade calculator I built in C++ that takes course information and calculates a semester GPA. 
I wanted to make something that felt more realistic instead of just working with random numbers, so this focuses on handling actual input and organizing data in a way that makes sense.
Compared to my other projects, this one pushed me to think more about structure and how to manage multiple pieces of information at once.
I had to work with vectors, format the output cleanly, and calculate GPA based on user input, which made it feel more like a real application.

It also helped me understand how to split my code across multiple files and keep everything organized while still handling more steps.

~Concepts Used
- Classes and object-oriented programming
- Vectors (dynamic arrays)
- User input handling
- Output formatting (iomanip)
- GPA calculation logic
- Multi-file program structure

~What the Program Does
- Asks the user for their name
- Takes input for multiple courses
- Stores course details (credits, grades, etc.)
- Calculates total credits and grade points
- Computes semester GPA
- Displays academic standing (Dean’s List, Good Standing, Probation)
- Prints a formatted grade report

~How to Run

1. Make sure you have a C++ compiler (like g++)
2. Open a terminal in the project folder
3. Compile the program:

g++ main.cpp -o gradesystem

4. Run the program:

./gradesystem

Follow the prompts to enter your courses and view your GPA report.
