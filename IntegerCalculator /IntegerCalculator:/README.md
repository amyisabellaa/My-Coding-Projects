~Integer Calculator

This project is a calculator I built in C++ using my own Integer class. Instead of just using normal integers, I wanted to understand how to actually create a class and control how operations like +, -, *, and / work behind the scenes.

Compared to my fraction project, this one was more about getting the basics down. I focused on understanding how objects interact, how operator overloading works in a simpler case, and how to structure my code in a way that actually makes sense.

At first it felt like I was just recreating something that already exists, but once it started working, it helped me understand how built-in types behave under the hood.

~Concepts Used
- Classes and object-oriented programming
- Operator overloading (+, -, *, /)
- Constructors
- Encapsulation
- Random number generation
- Program structure

~What the Program Does
- Generates random integers
- Performs random operations (+, -, *, /)
- Uses overloaded operators to calculate results
- Prevents division by zero
- Prints each calculation step clearly

~How to Run

1. Make sure you have a C++ compiler (like g++)
2. Open a terminal in the project folder
3. Compile the program:

g++ main.cpp Integer.cpp -o calculator

4. Run the program:

./calculator

The program will automatically generate and display random calculations each time you run it.
