~Matching Game

This is a memory matching game I built in C++ using a 4x4 board. This is one of my earlier projects.
I wanted to make something interactive instead of just basic programs, so this project focuses on actually playing a game while practicing core concepts.
I used this to get more comfortable with arrays, functions, and randomization, but also to understand how to structure a full program from start to finish. 
It was one of the first times I had to think about both the logic behind the code and how a user would interact with it.

~Concepts Used
  - 2D arrays to store the board
  - Functions to organize the program
  - Random shuffling so the board changes every time
  - User input validation
  - Game loop logic

~What the Program Does
  - Generates a 4x4 board with matching pairs
  - Shuffles the board each time the program runs
  - Lets the user pick coordinates to reveal cards
  - Checks if the selected cards match
  - Keeps going until all pairs are found
