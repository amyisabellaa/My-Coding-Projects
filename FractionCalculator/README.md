~Fraction Calculator

This project is a fraction calculator I built in C++ to get a better understanding of how classes and operator overloading actually work. 
Instead of just using regular numbers, I wanted to create my own type and control how operations like addition, subtraction, multiplication, and division happen.

At first, working with fractions in code was confusing, especially when it came to simplifying them and making sure everything stayed valid.
As I kept working on it, I started breaking the problem down step by step, and it helped me understand how everything connects behind the scenes.

~Concepts Used
- Classes and object-oriented programming
- Operator overloading (+, -, *, /)
- Constructors and copy constructors
- Encapsulation
- Random number generation
- Simplifying fractions using GCD

~What the Program Does
- Generates random fractions
- Performs random operations (+, -, *, /)
- Simplifies results automatically
- Prevents invalid cases like division by zero
- Displays each calculation clearly
